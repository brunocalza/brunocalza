## Hi there 👋

I'm a software engineer with a deep interest in databases and distributed systems. Some repositories you might find interesting:

- [go-bustub](https://github.com/brunocalza/go-bustub): An attempt to rewrite [cmu-db/bustub](https://github.com/cmu-db/bustub) in Go  
- [gossip-glomers](https://github.com/brunocalza/gossip-glomers): Solutions to [Fly.io’s Distributed Systems challenges](https://fly.io/dist-sys/)  
- [go-tableland](https://github.com/tablelandnetwork/go-tableland): A decentralized, EVM-based SQLite database  
- [sqlparser](https://github.com/tablelandnetwork/sqlparser): A SQL parser for `go-tableland`, written in `goyacc`  
- [pi-notes](https://github.com/brunocalza/pi-notes): My personal note-taking app  

I have a blog post where I write about things that interests me: [brunocalza.me](https://brunocalza.me).
